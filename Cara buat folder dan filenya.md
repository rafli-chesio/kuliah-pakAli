# 1. Cara membuat folder layout

Buka panel Project di sebelah kiri (biasanya tampilan Android).

Klik kanan pada folder res (Resources).

Pilih New > Android Resource Directory.

Pada bagian Resource type, pilih layout.

Klik OK. Sekarang folder layout akan muncul di bawah folder res.

# 2. Cara membuat file activity_main.xml

Klik kanan pada folder layout( yg diatas tadi) yang baru dibuat.

Pilih New > Layout Resource File.

Isi File name dengan activity_main (gunakan huruf kecil semua dan underscore, jangan pakai spasi).

Pada bagian Root element, kamu bisa mengetik androidx.constraintlayout.widget.ConstraintLayout agar sama dengan struktur yang kamu gunakan sekarang.

Klik OK.

# 3. Cara buat file second.kt

Klik kanan pada folder java > com.example.kuliah (atau nama packagemu).

Pilih New > Activity > Empty Views Activity.

Isi Activity Name, harus: second.

Pastikan centang Generate a Layout File sudah terisi agar file XML-nya juga ikut dibuat secara otomatis.

Klik Finish.
